# Stak - Full-Stack Scaffolding at your fingertips

Stak is a CLI tool which builds a full-stack application with a frontend, backend, and database language/framework of the user's choosing.

## Installation
#### Clone project from Git
```
git clone https://github.com/jgolden5/Stak
```
#### initialize project (this will install project into local bin directory and source the file)
```
./init.sh
```
#### Verify install
```
stak -v
```

## Scaffolding a full-stack app

```
stak -a js java postgres
```

## Scaffolding a frontend
```
stak -f js
```

## Scaffolding a backend
```
stak -b java
```

## Scaffolding a database
```
stak -d postgres
```
